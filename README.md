# Music Generation API (Meta MusicGen)

Профессиональное API для генерации музыки на базе Meta MusicGen, развернутое с использованием FastAPI, Clerk и RunPod.

## Архитектура "Aviation-Reliable"

Проект спроектирован с учетом строгого разделения ответственности:
- **FastAPI**: Асинхронный фреймворк для обработки HTTP-запросов.
- **Clerk**: Безопасная аутентификация корпоративного уровня.
- **RunPod**: Экономичное выполнение тяжелых ML-нагрузок на удаленных GPU.
- **Meta MusicGen**: Современная модель генерации аудио.

## Почему мы выбрали эти технологии?

### 1. RunPod (Экономическая эффективность)
Мы используем **RunPod Serverless** вместо постоянной аренды GPU. Это позволяет платить только за фактические секунды работы GPU во время инференса. 
- *Выгода*: Сокращение расходов до 90% при переменной нагрузке.
- *Масштабируемость*: Автоматическое масштабирование до сотен воркеров при росте очереди.

### 2. Meta MusicGen-Medium (Золотая середина)
Модель `musicgen-medium` выбрана как оптимальный баланс между качеством композиции (сопоставимым со сложными моделями вроде Suno) и скоростью генерации. Она позволяет получать высококачественные .wav файлы без задержек, свойственных более тяжелым архитектурам.

### 3. Clerk (Безопасность Enterprise-класса)
Безопасность реализована через валидацию JWT-токенов Clerk на уровне middleware.
- Проверка подписи с использованием JWKS.
- Изоляция пользовательских данных.
- Готовая инфраструктура для управления пользователями и ролями.

## Установка

1. Клонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настройте переменные окружения в файле `.env` (см. `.env.example`).

## Запуск

```bash
uvicorn main:app --reload
```

## API Эндпоинты

- `GET /health` — Проверка работоспособности системы.
- `POST /generate` — Генерация музыки.
  - **Auth**: Bearer Token (Clerk JWT)
  - **Body**:
    ```json
    {
      "prompt": "Lo-fi hip hop with calm piano melody",
      "duration": 15
    }
    ```

## Логирование
Система ведет детальные логи всех запросов, ошибок аутентификации и статусов задач в RunPod, что обеспечивает быструю диагностику в продакшене.
